# 前端设计文档

## 1. 设计概述

### 1.1 设计风格
**浪漫甜蜜风格** - 以粉色系为主，温馨可爱

### 1.2 设计原则
- **一致性**：统一的视觉风格和交互模式
- **简洁性**：界面清爽，信息层次清晰
- **响应式**：适配不同设备屏幕
- **可访问性**：支持键盘导航和屏幕阅读器

### 1.3 设计工具
- **Pencil**：手绘风格原型设计
- **Figma**：高保真 UI 设计（可选）
- **VS Code**：代码实现

## 2. 配色方案

### 2.1 主色调

```css
:root {
  /* 主色调 - 粉色系 */
  --primary-color: #ff6b9d;        /* 主粉色 */
  --primary-dark: #e55a8a;         /* 深粉色 */
  --primary-light: #ff9ff3;        /* 浅粉色 */
  
  /* 辅助色 */
  --secondary-color: #c44569;      /* 辅助粉色 */
  --accent-color: #ff9ff3;         /* 强调色 */
  
  /* 背景色 */
  --bg-color: #fff5f7;             /* 主背景 */
  --bg-alt: #fff0f3;               /* 辅助背景 */
  --bg-white: #ffffff;             /* 白色背景 */
  
  /* 文字色 */
  --text-color: #333333;           /* 主文字 */
  --text-light: #666666;           /* 浅色文字 */
  --text-white: #ffffff;           /* 白色文字 */
  
  /* 边框色 */
  --border-color: #ffd6e0;         /* 主边框 */
  --border-light: #ffe0e6;         /* 浅边框 */
  
  /* 阴影 */
  --shadow: 0 4px 6px rgba(255, 107, 157, 0.1);
  --shadow-lg: 0 10px 30px rgba(255, 107, 157, 0.2);
  
  /* 圆角 */
  --radius: 12px;
  --radius-sm: 8px;
  --radius-lg: 20px;
  --radius-full: 50px;
  
  /* 过渡 */
  --transition: all 0.3s ease;
}
```

### 2.2 配色应用场景

| 场景 | 颜色 | 说明 |
|------|------|------|
| 导航栏 | 白色背景 + 粉色文字 | 清爽、易读 |
| 按钮 | 粉色背景 + 白色文字 | 突出、可点击 |
| 卡片 | 白色背景 + 浅粉边框 | 内容分区 |
| 强调 | 深粉色 | 重要信息 |
| 背景 | 淡粉白 | 整体氛围 |

## 3. 字体设计

### 3.1 字体选择

```css
/* 主字体 */
font-family: 'Segoe UI', 'Microsoft YaHei', 'PingFang SC', sans-serif;

/* 等宽字体（代码） */
font-family: 'Consolas', 'Monaco', 'Courier New', monospace;
```

### 3.2 字号规范

| 级别 | 字号 | 行高 | 用途 |
|------|------|------|------|
| h1 | 4rem | 1.2 | 首页标题 |
| h2 | 2rem | 1.3 | 区域标题 |
| h3 | 1.2rem | 1.4 | 卡片标题 |
| body | 1rem | 1.6 | 正文内容 |
| small | 0.875rem | 1.5 | 辅助文字 |

### 3.3 字重规范

| 字重 | 数值 | 用途 |
|------|------|------|
| Regular | 400 | 正文 |
| Medium | 500 | 标题、按钮 |
| Bold | 700 | 强调文字 |

## 6. 动画设计

### 6.1 页面动画

```css
/* 入场动画 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes slideUp {
  from { transform: translateY(20px); opacity: 0; }
  to { transform: translateY(0); opacity: 1; }
}

@keyframes slideDown {
  from { transform: translateY(-20px); opacity: 0; }
  to { transform: translateY(0); opacity: 1; }
}

/* 粒子动画 */
@keyframes float {
  0%, 100% { transform: translateY(0) rotate(0deg); }
  50% { transform: translateY(-20px) rotate(180deg); }
}

/* 脉冲动画 */
@keyframes pulse {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.05); }
}

/* 弹跳动画 */
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10px); }
}
```

### 6.2 交互动画

```css
/* 按钮悬停 */
.btn:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(255, 107, 157, 0.3);
}

/* 卡片悬停 */
.card:hover {
  transform: translateY(-5px);
  box-shadow: 0 10px 30px rgba(255, 107, 157, 0.2);
}

/* 输入框聚焦 */
input:focus {
  border-color: var(--primary-color);
  box-shadow: 0 0 0 3px rgba(255, 107, 157, 0.1);
}
```

### 6.3 加载动画

```css
/* 旋转加载 */
.loading {
  animation: spin 1s linear infinite;
}

@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

/* 脉冲加载 */
.pulse {
  animation: pulse 1.5s ease-in-out infinite;
}
```

## 9. 状态设计

### 9.1 按钮状态

```css
/* 默认状态 */
.btn {
  background: var(--primary-color);
  color: white;
}

/* 悬停状态 */
.btn:hover {
  background: var(--primary-dark);
  transform: translateY(-2px);
}

/* 激活状态 */
.btn:active {
  transform: translateY(0);
}

/* 禁用状态 */
.btn:disabled {
  background: #ccc;
  cursor: not-allowed;
}
```

### 9.2 输入框状态

```css
/* 默认状态 */
input {
  border: 2px solid var(--border-color);
}

/* 聚焦状态 */
input:focus {
  border-color: var(--primary-color);
  box-shadow: 0 0 0 3px rgba(255, 107, 157, 0.1);
}

/* 错误状态 */
input.error {
  border-color: #e74c3c;
}

/* 成功状态 */
input.success {
  border-color: #27ae60;
}
```

### 9.3 卡片状态

```css
/* 默认状态 */
.card {
  background: white;
  box-shadow: var(--shadow);
}

/* 悬停状态 */
.card:hover {
  box-shadow: var(--shadow-lg);
  transform: translateY(-5px);
}

/* 选中状态 */
.card.selected {
  border: 2px solid var(--primary-color);
}
```
